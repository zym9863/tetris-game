# Tetris Mini Game

This project is a modern, user-friendly Tetris mini game implemented with HTML5, CSS3, and TypeScript. It features a stylish interface and smooth gameplay, suitable for both learning and entertainment.

## Directory Structure

```
├── src/
│   ├── index.html         # Main game page
│   ├── index.ts           # Entry point (TypeScript)
│   └── core/              # Game core logic
│       ├── game.ts            # Main game loop and state management
│       ├── renderer.ts        # Rendering logic
│       ├── tetromino-factory.ts # Tetromino factory
│       └── types.ts           # Type definitions
├── .gitignore
└── README.md
```

## Getting Started

1. Clone the repository and install dependencies:
```bash
git clone https://github.com/zym9863/tetris-game.git
cd tetris-game
npm install
```

2. Start development server:
```bash
npm run start
```

3. Build for production:
```bash
npm run build
```

## Features
- Modern color scheme and animation, stylish interface
- Classic Tetris gameplay with score, level, and line statistics
- Clear code structure for easy learning and extension
- Responsive layout for various screens
